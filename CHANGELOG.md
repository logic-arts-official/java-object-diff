# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.96] - 2024

### Repository Updates
- Repository moved to logic-arts-official organization
- Updated all documentation to reflect new repository location
- Refreshed documentation with current version information

### Dependencies
- SLF4J API 1.7.22
- Targets Java 5 bytecode (compatible with Java 5 and above)

### Documentation
- Updated all markdown documentation files
- Fixed broken links and outdated references
- Added comprehensive CHANGELOG
- Improved Getting Started guide
- Enhanced User Guide with better examples

## [0.95] - Previous Release

This version was the last release under the original repository.

### Features
- Complete object diffing functionality
- Support for collections and maps
- Circular reference detection
- Flexible configuration API
- Visitor pattern for traversing diff nodes
- Read and write access to object properties
- Merging capabilities

### Dependencies
- No runtime dependencies except SLF4J
- Compatible with Java 1.5 and above
